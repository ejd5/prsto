# ELTON OS — Routine quotidienne (15 minutes par jour)

## Objectif
Garder votre recherche executive active, organisée et efficace. Pas de configuration lourde, juste les actions qui comptent.

---

## 1. Ouvrir le Dashboard (1 min)

Ouvrir http://localhost:3000

Regarder la zone **"Aujourd'hui, quoi faire ?"** :
- Nouvelles offres détectées
- Top opportunités (score élevé)
- Candidatures à préparer
- Relances en attente

---

## 2. Vérifier les nouvelles offres (3 min)

- Cliquer **"Voir les nouvelles"** dans la zone Aujourd'hui
- Parcourir les offres récentes
- Pour chaque offre intéressante : cliquer **"Enregistrer"** (Shortlist)
- Pour les offres sans intérêt : cliquer **"Passer"**
- Les meilleures : cliquer directement **"Préparer candidature"**

---

## 3. Préparer 1 à 3 candidatures (5 min)

- Sur une offre shortlistée, cliquer **"Préparer candidature"**
- Attendre la génération (quelques secondes)
- ✅ Modal de succès ELTON OS s'affiche
- Cliquer **"Voir le dossier"**
- Vérifier rapidement :
  - CV adapté
  - Lettre de motivation
  - Email

---

## 4. Télécharger les documents (2 min)

- Ouvrir le dossier de candidature
- Aller dans l'onglet **CV** ou télécharger depuis **Documents**
- Choisir un template premium :
  - **Premium Leadership** — postes dirigeants (défaut)
  - **Executive Bordeaux** — timeline élégante
  - **Strategic Blue** — KPIs business
  - **Minimal Luxe** — épuré luxe
- Cliquer **Télécharger PDF** (pas de fenêtre print)
- Cliquer **Lettre** → Télécharger

---

## 5. Postuler sur le site (3 min)

- Ouvrir LinkedIn / Indeed / APEC
- Cliquer icône ELTON OS
- **Documents** → Rechercher le dossier → Joindre CV / Lettre
- **OU** télécharger le PDF et l'attacher manuellement
- **C'est VOUS qui cliquez sur Envoyer** — ELTON OS ne soumet jamais automatiquement

---

## 6. Mettre à jour le Pipeline (1 min)

- Aller dans **Pipeline**
- Changer le statut des candidatures :
  - Envoyé
  - Relance J+5
  - Entretien programmé
  - Offre reçue
- Planifier une **relance** si nécessaire (générée automatiquement)

---

## 7. Export hebdomadaire (1 min / semaine)

- Aller dans **Paramètres** → **Données**
- Cliquer **Export ZIP (Backup)**
- Sauvegarder le fichier `ELTON_OS_Backup_YYYY-MM-DD.zip`
- Ce fichier contient TOUTES vos données sauf les clés API
- Conservez-le en sécurité (cloud, disque externe)

---

## Ce que ELTON OS fait automatiquement

- ✅ Import des offres depuis les sources configurées
- ✅ Scoring des offres (compatibilité avec votre profil)
- ✅ Génération des documents (CV + lettre + email)
- ✅ Détection des doublons

## Ce que ELTON OS ne fait jamais automatiquement

- ❌ Ne postule pas à votre place
- ❌ N'envoie pas d'email
- ❌ Ne scrape pas LinkedIn
- ❌ Ne contourne pas les CAPTCHA
- ❌ Ne stocke pas vos cookies
- ❌ N'envoie pas de données dans le cloud
