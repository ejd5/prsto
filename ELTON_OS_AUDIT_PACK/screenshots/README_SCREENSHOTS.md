## Capture Instructions

To capture screenshots manually:
1. Start the app: `npm run dev`
2. Open http://localhost:3000
3. Navigate to each screen listed above
4. Take full-page screenshots (browser dev tools or dedicated tool)
5. Name files as listed above
6. Place PNG files in this directory

## Important
- Do NOT capture screenshots containing real personal data (emails, phone numbers, real names)
- If the database contains real data, use demo mode or clear data before capturing
- Prefer using `npm run db:seed` to populate with demo/non-sensitive data before screenshotting
