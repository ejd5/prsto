## Résumé des correctifs

| Problème | Correctif |
|----------|-----------|
| Footer ELTON OS dans les PDF | Supprimé de tous les générateurs et templates |
| Formation affichée en JSON brut | Nettoyé par `sanitizeCvForExport()` |
| Langues sans niveaux | `normalizeLanguages()` utilisé |
| Expertises dupliquées (CRM / Salesforce CRM) | `normalizeSkills()` fusionne les groupes |
| Résumé absent dans certains PDF | `extractSummary()` utilise `profile.summary` comme fallback |
| Template non respecté par extension | Template transmis explicitement via `?template=` |
| Dashboard filters changeaient les compteurs | `allJobs` / `displayedJobs` séparés |
