# Concepts Landing Page ELTON OS

## Concept 2 — PILOT
Ouvre `02-pilot.html` ou `02-pilot-complet.html` dans le navigateur.

Style : Claire, moderne, SaaS B2B, dégradé bleu/teal.
Ambiance : Professionnelle, rassurante, « ça inspire confiance ».
Pour : Directeurs qui veulent du sérieux et de la clarté.

## Concept 3 — STRAIGHT
Ouvre `03-straight.html` ou `03-straight-complet.html` dans le navigateur.

Style : Minimaliste, audacieuse, typographie forte, noir/blanc.
Ambiance : Éditoriale, « on ne fait pas comme tout le monde ».
Pour : Marque forte, positionnement différenciant.

---

Ouvrir dans le navigateur :
```bash
open "02-pilot-complet.html"
open "03-straight-complet.html"
```
